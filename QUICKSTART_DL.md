# ⚡ 深度學習應用 - 快速開始指南

**目標：5 分鐘內啟動並運行深度學習應用**

---

## 🎯 三步啟動

### 步驟 1️⃣：安裝依賴（2 分鐘）

```bash
pip install -r requirements.txt
```

或使用 conda：

```bash
conda create -n deep_learning python=3.9
conda activate deep_learning
pip install -r requirements.txt
```

**預期輸出：**
```
Successfully installed torch, tensorflow, streamlit, pandas, numpy, ...
```

### 步驟 2️⃣：啟動應用（1 分鐘）

```bash
streamlit run deeplearning_app.py
```

**預期輸出：**
```
You can now view your Streamlit app in your browser.

  Local URL: http://localhost:8501
  Network URL: http://192.168.1.x:8501
```

應用會自動在瀏覽器打開（通常在 `http://localhost:8501`）。

### 步驟 3️⃣：按照流程操作（2 分鐘）

1. **選擇階段** - 左側邊欄選擇 CRISP-DM 階段
2. **跟隨提示** - 按照應用的步驟操作
3. **查看結果** - 實時查看可視化和指標

## 💡 常見任務

### 加載自己的數據

1. 進入「2️⃣ 數據理解」階段
2. 選擇「上傳 CSV 文件」
3. 上傳你的 CSV 文件
4. 點擊「生成數據」

### 改變模型架構

1. 進入「4️⃣ 建模」階段
2. 在「隱藏層大小」中選擇不同的值
3. 調整「批大小」和「訓練週期」
4. 點擊「開始訓練」

### 查看詳細的訓練過程

1. 模型訓練中會顯示進度條
2. 訓練完成後查看「訓練損失曲線」
3. 使用不同的配置進行多次實驗對比

### 保存最好的模型

1. 進入「6️⃣ 部署」階段
2. 點擊「保存為 PyTorch 格式」
3. 模型會保存到 `models/` 目錄

## 🔍 故障排查快速參考

| 問題 | 解決方案 |
|------|---------|
| `ModuleNotFoundError: No module named 'torch'` | 運行 `pip install -r requirements.txt` |
| `CUDA 內存不足` | 減少批大小或隱藏層大小 |
| `應用啟動緩慢` | 首次加載需要 30-60 秒，之後會快 |
| `模型不收斂` | 增加訓練週期或調整學習率 |
| `過擬合` | 增加 Dropout 或減少模型大小 |

## 🚀 性能優化提示

### 加速訓練

1. **使用 GPU**
   ```bash
   # 在模型層自動選擇
   # device = 'cuda' if torch.cuda.is_available() else 'cpu'
   ```

2. **增加批大小**
   - 更大的批 = 更快的訓練（但需要更多內存）

3. **減少訓練週期**
   - 使用早停機制避免不必要的訓練

### 減少內存使用

1. **減少批大小** - 8 或 16 而不是 32
2. **減少隱藏層大小** - [64, 32] 而不是 [256, 128]
3. **減少模型深度** - 2-3 層而不是 5 層

## 📞 快速幫助

**應用不啟動？**
- 檢查 Python 版本：`python --version`
- 驗證依賴：`pip list`
- 查看錯誤信息：運行時會顯示

**模型訓練卡住？**
- 減少批大小
- 減少訓練週期
- 檢查 GPU 內存使用

**結果看起來不對？**
- 檢查數據質量
- 驗證特征縮放
- 嘗試不同的模型架構
