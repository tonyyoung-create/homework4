# 🎯 深度學習項目 - 項目開始指南

## 📌 項目概述

你已經獲得了一個**完整的深度學習應用**，遵循 **CRISP-DM** 方法論，具有以下特點：

### ✨ 核心特點

- ✅ **完全本地運行** - 無需 API、無需網絡、無需 GPU
- ✅ **遵循 CRISP-DM** - 業界標準的 6 階段工作流
- ✅ **深度學習框架** - PyTorch（可選 TensorFlow）
- ✅ **交互式界面** - Streamlit Web UI
- ✅ **端到端工作流** - 從數據到部署

## 🚀 5 分鐘快速開始

### 第 1 步：安裝依賴（2 分鐘）

```bash
# 確保在項目目錄中
cd 作業4

# 安裝依賴
pip install -r requirements.txt
```

**如果第一次安裝 PyTorch，可能需要 5-10 分鐘。**

### 第 2 步：啟動應用（1 分鐘）

```bash
streamlit run deeplearning_app.py
```

應用會自動打開瀏覽器，顯示 `http://localhost:8501`

### 第 3 步：按照 6 個階段操作（2 分鐘）

1. **業務理解** - 了解項目目標
2. **數據理解** - 加載/生成數據
3. **數據準備** - 預處理數據
4. **建模** - 訓練模型
5. **評估** - 查看性能
6. **部署** - 保存模型

## 🆘 快速故障排查

| 問題 | 解決方案 |
|------|---------|
| `ModuleNotFoundError: No module named 'torch'` | 運行 `pip install torch` |
| 應用啟動緩慢 | 首次加載需要時間，耐心等待 |
| GPU 不被識別 | 正常情況下使用 CPU，性能仍可接受 |
| 訓練卡住 | 減少批大小或訓練週期 |

**詳細故障排查：** 查看 `QUICKSTART_DL.md`
