🆘 快速幫助
───────────────────────────────────────────────────────────────────────────────

問題：安裝失敗
解決：確保 Python 3.8+，運行 pip install --upgrade pip

問題：應用啟動緩慢
解決：首次加載需要時間（30-60 秒），耐心等待

問題：GPU 不被識別
解決：正常，應用自動使用 CPU，性能仍可接受

問題：模型訓練卡住
解決：減少批大小或訓練週期

更多幫助見：QUICKSTART_DL.md（故障排查部分）
